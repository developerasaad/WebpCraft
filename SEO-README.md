## ğŸ› Troubleshooting

### Schema Not Detected?
1. Verify page is publicly accessible
2. Check JSON-LD syntax with validator
3. Ensure schema is in `<head>` section
4. Wait 24-48 hours for Google to re-crawl

### Rich Results Not Showing?
1. Verify schema passes Rich Results Test
2. Ensure content is visible on page
3. Check for duplicate schemas
4. Wait 1-2 weeks for processing

### Build Errors?
1. Run `npm run build` to check for errors
2. Validate TypeScript types
3. Check schema.ts for syntax errors
4. Review console for warnings
